Learning Path
Phase 0: Starting from zero
Topics: What is Linux, What is Networking, What is Programming, What is Cloud Computing, What is DevOps, What is a Cloud Engineer.

Topics: Version Control, Cloud CLI, Infrastructure as Code, SSH, CLI Basics, CTF Lab.
Phase 2: Programming & AI Integration
Topics: Python, APIs, FastAPI, Databases, GenAI APIs, Prompt Engineering, Build the App.

Phase 4: DevOps Fundamentals

Phase 5: Securing Your Cloud Applications
Topics: Identity and Access Management, Data Protection & Secrets, Network Security, Security Monitoring, Threat Detection & Response, Capstone Project.
